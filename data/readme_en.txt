----------------------------------------------------------------
■  Included files
----------------------------------------------------------------

  ・readme.txt                     ... This file
  ・viz_pos.html                   ... Generated result rendering HTML
  ・trace_model/trace_model.pmx    ... generation model (with mesh)
  ・xxx                            ... Generated result folder
      ・xxx.json                   ... JSON file
      ・xxx.avi                    ... motion video
      ・trace_noXXX.pmx            ... A stick figure model with the same skeleton as the source model
      ・trace_noXXX_original.vmd   ... all-hitting motion
      ・trace_noXXX_thinning.vmd   ... thinned motion

  ・The number of json and vmd is matched with the generated number
